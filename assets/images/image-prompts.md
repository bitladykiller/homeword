# 小葵的污染体检报告 - 图片生成 Prompt 清单

## 设计风格统一要求
- 整体风格：扁平插画 + 微质感，科普绘本感

- 背景：干净简洁，留白充足
- 所有图片统一为 16:9 或 1:1 比例，PNG透明背景或浅色背景

## 使用建议

### 必须替换的CSS图形（优先级高）
1. hero-sunflower.png - 首页主视觉，影响第一印象
2. leaf-healthy.png / leaf-damaged.png - 叶片对比页核心内容
3. root-healthy.png / root-damaged.png - 根系对比页核心内容
4. photosynthesis-diagram.png - 光合作用页核心图示

### 强烈建议替换（优先级中）
5. air-pollution-scene.png - 空气污染页视觉
6. water-pollution-scene.png - 水体污染页视觉
7. soil-pollution-scene.png - 土壤污染页视觉
8. xiaokui-avatar.png - 全站吉祥物，增强品牌感

### 锦上添花（优先级低）
9. plant-reed.png / sunflower.png / vetiver.png / willow.png - 植物图鉴页
10. icon-air/water/soil-control.png - 治理页图标
11. action-bg.png - 行动页背景
12. decoration-leaves.png - 全站装饰元素
13. report-stamp.png - 体检报告印章

### 图片优化建议
- 所有图片导出为 PNG 格式（需要透明背景时）或 JPG（不需要透明时）
- 单张图片大小控制在 200KB 以内
- 使用 tinypng.com 等工具压缩
- 为每张图片添加合适的 alt 文本
